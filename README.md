# SimpleSTAC

SimpleSTAC eases the use of pystac.

# Install

Within a mamba env:
```shell
mamba env create -f https://gitlab.irstea.fr/umr-tetis/stac/simplestac/-/raw/main/environment.yml

```

Within a virtualenv:
```shell
pip install git+https://gitlab.irstea.fr/umr-tetis/stac/simplestac
```
